# **App Name**: TravelSphere

## Core Features:

- Explore & Discover Packages: Browse a hero section with search filters (destination, budget), featured packages, and special offers on the home page. View a dynamic list of all packages with filters for price range and category.
- Detailed Package View: View comprehensive details for each travel package, including a day-by-day itinerary, 'What's Included' list (hotel, food, guide), and a persistent booking sidebar.
- User Authentication: Enable user registration and login securely via Firebase Email/Password authentication.
- Personalized User Profile: Authenticated users can access a 'My Profile' page to view their booking history cards.
- Streamlined Checkout Process: Navigate a multi-step payment page including a booking summary and mock payment options (UPI, Card, Net Banking).
- Support & Information Hub: Access essential information through a Contact Form, an 'About Us' section, and an accordion-style FAQ page covering refunds and cancellations.
- AI-Powered Travel Suggestion Tool: Utilize an AI tool to generate personalized travel tips or itinerary suggestions based on selected package details or user preferences, enhancing trip planning.

## Style Guidelines:

- Primary Color: Vibrant Teal (#00A699) to evoke freshness and professionalism, aligning with travel themes. (HSL: 175, 100%, 32.5%)
- Background Color: A light, desaturated teal (#E1F5F4) providing a clean and expansive canvas for content, fitting the chosen light color scheme. (HSL: 175, 20%, 95%)
- Accent Color: A bright, cheerful spring green (#8CE052) for calls to action and highlights, creating good visual contrast with the primary teal. (HSL: 145, 70%, 60%)
- Headline and Body Font: 'PT Sans' (sans-serif) for its modern, clean appearance with a touch of warmth, ensuring readability across all content, from package descriptions to FAQs.
- Utilize 'Lucide-react' for crisp, modern icons to represent categories and actions consistently throughout the application.
- Responsive grid-based layout ensuring optimal viewing experience across all devices, with a focus on generous spacing and clear content hierarchy.
- Subtle and elegant transitions for page navigation and component interactions, enhancing the modern and clean user experience without distraction.
